# meikyuInTerminal

ターミナルで行う簡単な謎解きゲーム
Mac(ターミナル)、Win(Powershell)で動作
## 使用コマンド
| コマンド       | 説明                     | 
| -------------- | ------------------------ | 
| ls             | 部屋の中を確認           | 
| ls -a (-Force) | 隠し部屋やノートを探す。 | 
| cat 〇〇.txt   | テキストを確認           | 
| cd 〇〇        | 部屋を移動               | 
| pwd            | いまどこにいるかを確認   |
